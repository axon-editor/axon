## Axon v1.0.5

### Search

- Tightened workspace search exclusions so generated Go cache folders stay out of results.
- Added path-segment filtering for .gocache, gocache, and go-build folder variants.
- Added regression coverage so cache folders do not show up as source search hits again.

### Tabs

- Improved truncated pane and terminal tab spacing so long names do not crowd the close button.

### Command Palette

- Added grouped command metadata and shortcut hints.
- Improved ranking with exact, prefix, substring, and fuzzy matching.
- Added Page Up, Page Down, Home, and End navigation.
- Added palette actions for active HTML preview and update notes.
