# Axon v1.0.4

This release tightens the updater flow and keeps workspace loading calmer when you switch projects.

## Highlights

### Update Flow

- Download new releases in the background
- Show `Downloading xx%` in the toolbar while the update is staged
- Keep the release notes modal closed until the user clicks the update button
- Switch the primary action to `Restart to Update` once the update is ready
- Hand off restart and relaunch to the updater after the user confirms

### Workspace Behavior

- Keep root folders collapsed by default when a workspace is opened or switched
- Avoid auto-expanding the entire first level of the tree on fetch
- Add `.gocache/` to the repo ignore list so local Go build caches do not pollute the workspace

## Notes

- This is a hotfix release line in v1, meant to be downloaded manually first.
- The next release can be used to validate the in-app restart/update path end to end.
