# Axon v1.0.0

Axon v1 is the first packaged desktop release of the editor.

## Highlights

- Desktop packaging for macOS, Windows, and Linux
- GitHub Release workflow with release artifacts
- Multi-pane editor with tabs, splits, drag/drop, and dirty file handling
- Custom title bar, splash screen, settings UI, themes, and font controls
- Built-in Markdown preview with HTML/style support
- Terminal panel with tabs, theme support, and persistent sessions
- Git file status, diff modal, changed-file indicators, and gutter markers
- Workspace search with jump-to-line behavior
- Command palette foundation for editor actions
- Problems, output, and language-server groundwork
- Sidebar context menus, file/folder creation, rename, delete, and indentation guides

## Downloads

Use the artifact for your platform:

- macOS: .dmg or .zip
- Windows: .exe
- Linux: .AppImage or .deb

## Notes

- The app is currently unsigned, so macOS may show a security warning on first open.
- The Go core service is still separate in this v1 setup. Terminal-backed features may need the core service running locally.
- AI features are intentionally not included yet. The editor foundations were built first so the AI feature can use real project context later.
