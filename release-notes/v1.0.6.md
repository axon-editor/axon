## Axon v1.0.6

### LSP Completions

- Added LSP-backed completion requests through Axon's main-process language server bridge.
- Added JSON-RPC response handling for language server stdout so Axon can request completions, not only start server processes.
- Added active-language server startup so opening Rust, C++, Go, Python, TypeScript, or JavaScript files can start the matching server.
- Added Monaco completion providers for TypeScript, JavaScript, Go, Rust, Python, and C++.
- Synchronized dirty in-memory editor text with language servers before requesting completion results.

### Web Editing

- Added immediate HTML and JSX/TSX tag snippets so common tags like div, button, section, input, and form appear while language servers warm up.
- Enabled quicker suggestion popups, snippet-first ordering, and tab completion in the editor.

### Update Testing

- Bumped Axon to 1.0.6 so installed v1.0.5 builds can detect a newer release for update-flow testing.

### Notes

- Rust and C++ completions require rust-analyzer and clangd.
- Go, Python, and TypeScript project completions require gopls, pyright-langserver, and typescript-language-server respectively.
