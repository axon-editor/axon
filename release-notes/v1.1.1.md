## Axon v1.1.1

### Stable Dev Startup

- Locked the Vite development server to port `5173` so Axon no longer drifts to
  another port while Electron keeps loading the original dev URL.
- Added an explicit Electron development server URL so the main process and the
  dev script stay aligned.
- Added a development-only single-instance guard so running `npm run dev` more
  than once focuses the current Axon dev app instead of leaving multiple
  Electron apps in the Dock.

### Terminal PATH

- Added NVM Node version discovery to Axon's integrated terminal environment.
- New terminal sessions now include installed NVM Node `bin` folders in PATH,
  which keeps `node` and `npm` available even when Axon is launched from a GUI
  process instead of a fully initialized shell.
