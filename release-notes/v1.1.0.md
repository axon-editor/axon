## Axon v1.1.0

### Bundled Extensions

- Added bundled Axon theme extensions that ship with the app instead of only loading from the current workspace.
- Added Arctikai as an Axon default theme.
- Added Anysphere as an Axon default theme.
- Added the Apathy theme family as Axon defaults: Apathy, Apathetic Ocean, and Minted.
- Included bundled extension files in the packaged Electron app so the themes work in release builds.

### Fonts

- Bundled a curated Monaspace Nerd Font set for editor and terminal use.
- Added Monaspace Neon NF, Argon NF, Krypton NF, Radon NF, and Xenon NF.
- Included core weights from ExtraLight through ExtraBold so lighter presets like Apathy Ocean can render properly.
- Added Monaspace Nerd Font presets in settings.

### Extension Reliability

- Hardened extension theme reload so invalid token styles cannot crash Monaco theme registration.
- Kept bundled themes in the same manifest shape as user extensions so future default extensions can reuse the same loader.

### Verification

- `npm run build:main`
- `npm run build:renderer`
- `git diff --check`
