## Axon v1.0.2

This release is mainly for testing the in-app updater with a real patch version while also fixing editor polish that affects daily use.

### Added

- Added the full Catppuccin Latte icon asset set to the packaged app.
- Added broader file, folder, extension, and config-file icon mappings so common project files no longer fall back to the generic icon.
- Added dedicated handling for `.env` variants, including `.env.local`, `.env.production`, and other `.env.*` files.

### Fixed

- Fixed the active Git compare modal so the current side refreshes live from the Monaco editor model while you type.
- Fixed packaged renderer asset loading for `axon.png` by moving it into the shared public asset directory.
- Fixed native macOS Quit crashing when Electron tried to read a destroyed `BrowserWindow.webContents`.

### Changed

- Moved Catppuccin icon resolution into a dedicated mapping module so the renderer icon component stays small and easier to maintain.
- Pointed Vite at Axon's shared public directory so clean release builds include icons, the app logo, and updater-facing packaged assets consistently.

### Notes

- macOS builds are still unsigned, so Gatekeeper may warn before opening the app.
- This release includes updater metadata for Windows, macOS, and Linux so the in-app updater can detect and download the new version.
