## Axon v1.1.10

Axon v1.1.10 is a broad polish and hardening release. It tightens workspace
trust, improves font handling, adds cursor controls, refines settings UI
behavior, fixes Git file-state coloring for media paths, and keeps the app blur
and transparent theme layering aligned.

### Workspace trust

- Reworked workspace trust into a real execution gate instead of a passive UI
  flag.
- Untrusted workspaces no longer start terminals, tasks, language servers,
  HTML preview, or extensions.
- Project LSP bridge paths are hidden from the renderer while a workspace is
  untrusted, which keeps hover, completion, rename, and other project-backed
  features from running when the folder has not been trusted.
- Revoking trust now uses a custom Axon confirmation modal instead of the
  native confirm dialog.
- The AI sidebar now stays closed on startup instead of opening by default.

### Fonts

- Added local font discovery for the bundled NerdFonts folder so Axon can
  surface the downloaded Monaspace Nerd families automatically.
- Font dropdowns now preview each option using its own family.
- Imported fonts still work through Axon storage, and the dropdown now shows
  both imported fonts and local NerdFonts entries without forcing users to
  import every file manually.
- Font metadata now preserves weight, style, and stretch so bundled NerdFont
  variants register correctly instead of turning into noisy separate families.

### Cursor controls

- Added cursor style controls to the editor settings.
- Added cursor blinking controls to the editor settings.
- Monaco now follows the saved cursor style and blinking preference directly,
  including the `expand` blinking mode.

### Settings UI

- Removed the decorative traffic-light row from the settings modal.
- Kept the flatter Zed-style settings layout with a plain sidebar and simple
  section rows.
- Settings rows stay compact and readable instead of looking like stacked
  cards.

### Git and files

- Fixed Git status parsing so filenames with spaces are handled correctly.
- Media files and screenshots now receive the same file-name status coloring as
  normal text files.
- Removed the extra icon outline/badge treatment around changed files and kept
  the Git status visible on the name itself.
- Sidebar create/rename/move/delete flows continue to refresh immediately and
  keep expanded trees in sync with local operations.

### Transparency and blur

- Adjusted the transparent-app layering so the theme no longer paints on top of
  the blur effect.
- Extended transparency to more app surfaces, including sidebar, panel, tab,
  gutter, and terminal backgrounds.
- The app blur now reads as an actual background effect instead of blurring the
  whole app chrome.

### Reliability

- Preserved the existing terminal, LSP, and settings hardening work across dev
  and packaged builds.
- Kept the release build checks aligned with the same runtime paths used in
  development so the packaged app stays close to the working editor state.

### Notes

- Full `/NerdFonts` assets are not bundled into the release by default because
  the directory is very large. Axon scans the folder locally when it exists and
  also ships the committed Monaspace Nerd bundle in `editor/public/fonts`.
- Workspace trust in Axon is intentionally strict for project execution paths.
  Basic text editing still works, but code-running surfaces stay disabled until
  the workspace is trusted.
