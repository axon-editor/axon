## Axon v1.1.9

Axon v1.1.9 is a hotfix release focused on keeping the packaged app aligned
with the behavior already working in dev.

### Terminal

- Fixed terminal output replay so long-running sessions do not lose visible
  text when xterm is still catching up.
- Kept multiline terminal input on a bracketed-paste path so `Shift+Enter`
  behaves like a real newline instead of a premature submit.
- Increased the core terminal replay window and websocket write timeout so
  chatty shells and AI prompts stay stable for longer sessions.

### Packaged LSP

- Verified the packaged app can launch Node-backed language servers from
  `app.asar` instead of depending on unpacked duplicate trees.
- Added an explicit runtime dependency for GraphQL so GraphQL language-server
  startup works in the packaged app.
- Patched the MDX markdown language-server dependency so packaged Electron Node
  can load `vscode-uri` correctly.
- Kept the release verifier checking the full language-server dependency
  closure before packaging, which catches missing runtime modules earlier.

### Notes

- This release keeps the release process safe for dev and packaged Axon by
  focusing on production-only failures.
- The bundled language-server set is still intentionally large because Axon
  now ships the editor languages out of the box instead of asking users to
  install them one by one.
