## Axon v1.0.7

### LSP foundation

- Adds the five-language LSP foundation for TypeScript/JavaScript, Go, Python, Rust, and C/C++.
- Bundles TypeScript/JavaScript language-server support inside the packaged editor.
- Adds live LSP diagnostics into the Problems panel.
- Syncs dirty editor buffers to language servers so diagnostics follow unsaved changes.
- Keeps instant local-symbol suggestions so autocomplete appears quickly while LSP results load.

### Markdown preview links

- Opens external web links in the default browser.
- Opens mailto links in the default mail app.
- Opens tel links through the system phone handler when available.
- Opens local Markdown links inside Axon instead of navigating/replacing the app window.

### Docs

- Updates the root README, editor README, and core README.
- Adds CHANGELOG.md.
- Adds docs for updates/releases and language servers.

### Notes

- macOS builds are still unsigned personal builds, so fully automatic in-app replacement may require manual download/replacement until Axon is signed and notarized.
- Go, Python, Rust, and C/C++ intelligence depends on local language-server/toolchain installs: gopls, pyright-langserver, rust-analyzer, and clangd.
