## Axon v1.0.1

This release focuses on markdown preview polish, Source Control usability, workspace empty states, and the first update-check test release after v1.0.0.

### Changed

- Markdown fenced code blocks now render as rounded, bordered code panels for command examples.
- Inline markdown code now stays compact, so text like `7777` and `message here` renders as small code pills instead of full blocks.
- Markdown code copy now falls back to Electron clipboard and shows a checked success state after copying.
- Source Control now uses a dedicated fixed modal with separate scroll areas for the changed-file list and diff preview.
- Empty editor space now separates the no-folder welcome pane from the folder-open blank workspace state.
- New windows can start as clean sessions instead of cloning an already-open workspace session.
- Added a better workspace loading overlay during folder open and restore.
- Added font stack aliases and improved default editor/terminal readability.
- Gitignored files and folders are dimmed in the sidebar while real Git changes keep their status colors.

### Release

- Bumped Axon to 1.0.1 so installed v1.0.0 builds can detect this release through the update prompt.
