# Axon v1.0.3

This release adds built-in HTML preview support, makes workspace loading lighter, and fixes the updater restart flow so the app relaunches cleanly after installation.

## Highlights

### HTML Preview

- Preview `.html` and `.htm` files directly in Axon
- Live reload while editing HTML, CSS, JS, and linked assets
- Open previews in your default browser
- Device presets for responsive, phone, tablet, and desktop views
- Console output and runtime error capture inside the preview panel
- Dedicated preview tabs so the source file and preview can stay open together

### Workspace Loading

- Switched the file tree to lazy folder loading
- Folders now fetch their children only when expanded
- Faster workspace open time on larger projects

### Editor Reliability

- Added Monaco worker wiring for Vite/Electron so language features stay off the main thread
- Added a renderer CSP to reduce security warnings in development
- Hardened update bootstrapping when the renderer and main process are out of sync during local development

### Updater Fix

- Fixed the restart path so installing an update no longer quits the app without relaunching it

## Downloads

- macOS Intel: `.zip`
- macOS Apple Silicon: `.zip`
- Windows: `.exe`
- Linux AppImage: `.AppImage`
- Linux DEB: `.deb`

## Notes

- The update flow is still packaged-release driven.
- If you are on macOS, use the correct architecture download for your machine.
- The release page now matches the `v1.0.3` tag.
