## Axon v1.2.2

Axon v1.2.2 focuses on making Axon Agent feel like a real local editor
assistant instead of a prompt box with a large context dump. The agent can now
ask the backend for project files, searches, and file reads, while the UI starts
the stream without waiting for a heavy project-context preload.

### Project-aware Axon Agent tools

- Added backend project tools in `axon-core` for listing files, reading files,
  and searching project text.
- Kept tool execution inside the current workspace so model requests cannot
  read arbitrary filesystem paths outside the open project.
- Skipped dependency folders, build output, cache folders, binary files, and
  generated files when tools search or read project content.
- Added a fallback project probe for smaller local models that sometimes answer
  from guesses instead of calling tools.

### Faster chat startup

- Removed renderer-side project context preloading from the Ask Axon submit
  path.
- Let the backend tools fetch only the files or matches needed for the current
  question.
- Kept active-file, diagnostics, Git changes, and conversation context in the
  request while avoiding a large full-project prompt before streaming starts.

### Local models runtime startup

- Axon now warms up the local Axon models runtime when the app opens.
- If the local runtime is installed but not running, Axon asks `axon-core` to
  start it in the background.
- Packaged macOS builds now check common Ollama install paths such as
  `/opt/homebrew/bin/ollama`, `/usr/local/bin/ollama`, and the Ollama app bundle
  resource path when the GUI app `PATH` does not include them.

### Conversation sessions

- Ask Axon conversations are now stored as workspace-scoped sessions.
- The conversation picker can switch between previous conversations.
- The new conversation button creates a fresh thread without deleting older
  conversations.
- Each conversation can be cleared from the picker after a confirmation modal.
- Clearing one conversation keeps the other saved conversations intact.

### UI structure

- Split Ask Axon message rendering, conversation picking, and clear-confirmation
  UI into smaller components.
- Kept the main Ask Axon sidebar below the project file-size cap while adding
  the new conversation controls.
