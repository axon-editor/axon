## Axon v1.0.9

### LSP UX

- Added LSP signature help so function parameters can appear while typing calls.
- Added LSP code-action plumbing through the main-process language-server bridge and Monaco provider system.
- Added Docker and Tailwind CSS to the Workspace Servers list.
- Added Dockerfile snippets for common instructions like `FROM`, `WORKDIR`, `COPY`, `RUN`, `CMD`, `EXPOSE`, `ENV`, and `ARG`.
- Added Tailwind utility suggestions for HTML, CSS, JSX, and TSX class contexts.
- Fixed HTML element snippets so tags like `div` no longer appear inside `class`, `className`, or `style` values.

### Bundled Language Servers

- Bundled `@tailwindcss/language-server` so Tailwind support works without a global install.
- Bundled `dockerfile-language-server-nodejs` so Dockerfile support works without a global install.
- Docker and Tailwind servers now resolve from project `node_modules/.bin` first, bundled Axon dependencies second, and global PATH last.
- Packaged builds unpack Tailwind native watcher files so the bundled server can load them reliably.

### Search

- Added replace controls to workspace search.
- Added replace-selected and replace-all-visible flows using Axon's existing file read/write path.
- Added local workspace search history for faster repeated searches.

### Source Control

- Added batch stage and batch unstage actions.
- Added discard-all for unstaged changes with confirmation.
- Added a commit message box for committing staged changes from Source Control.
- Commits run through the main process with `git commit -F -` so multi-line messages and quotes are preserved safely.

### Appearance

- Added Cascadia Code and Berkeley Mono font presets.
- Expanded syntax token coverage for function, method, member, identifier, parameter, JSON/YAML key, and Dockerfile token scopes.

### Verification

- `npm run build:main`
- `npm run build:renderer`
- `git diff --check`
