## Axon v1.0.8

### Appearance Engine

- Added a real theme system with separate Axon Dark, Sora, Zed Dark, Catppuccin Mocha, and Ayu Dark modules.
- Theme definitions now own UI tokens and Monaco syntax tokens together.
- Removed the hardcoded default Ayu Dark override. User overrides now layer cleanly on top of built-in themes.
- Added richer syntax colors for comments, strings, numbers, keywords, functions, methods, classes, types, interfaces, variables, constants, operators, brackets, imports, JSX tags, and attributes.
- Added stronger default-token fallbacks so Markdown, go.mod, go.sum, .sim, plaintext, and other low-token files do not render black text on dark editor backgrounds.
- Added Go function and method coloring fallback for Monaco's basic Go tokenizer.
- Added font presets, font weight, and live typography updates across the app shell, editor, diff view, and terminal.
- Prevented the editor theme flash by waiting for saved settings to load before mounting editor panes.

### LSP Navigation

- Added LSP hover cards through Monaco.
- Added LSP-backed go-to-definition and find-references providers.
- Added rename symbol and format document commands.
- Added typed IPC contracts for hover, definitions, references, rename, formatting, diagnostics, and document sync.
- Kept language-server payloads normalized in the main process before sending them to the renderer.

### Fixes

- Fixed Workspace Servers Start, Stop, and Refresh so checking language-server availability no longer launches Electron helper windows or reloads the app.
- Hardened macOS window lifecycle handling so quit, activate, and update flows do not restore stale sessions or create surprise windows.
- Blocked browser reload and DevTools shortcuts in the packaged app and development builds so Axon behaves like a desktop editor.
